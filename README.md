# My dotfiles
These are my dotfiles. I'll get around to organizing these eventually.

## Current config
A list of what is currently configured with these files:

- [Rust](https://www.rust-lang.org/) - Changing the default linker to [Mold](https://github.com/rui314/mold).
- [i3](https://i3wm.org/) - My window manager
- [alacritty](https://alacritty.org/) - My terminal
- [nnn](https://github.com/jarun/nnn) - Terminal file manager
- [neovim](https://neovim.io/) - My terminal text editor
- [picom](https://github.com/yshui/picom) - Linux X11 compositor
- [polybar](https://polybar.github.io/) - My task bar
- [fish](https://fishshell.com/) - My terminal shell
- [tmux](https://github.com/tmux/tmux) - Terminal multiplexer
- `.xinitrc` - X11 initial configuration
- `.Xresources` - X11 resources configuration
